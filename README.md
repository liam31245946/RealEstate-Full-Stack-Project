This is my first ever full stack project. I created a Real Estate App for people to come and view the avaiable properties, join as buyer or seller. 
Features:
+ Googe Map
+ Upload and Update/Delete as seller
+ Save as favorite and view them later as Buyer or Seller
+ Filter out properties
